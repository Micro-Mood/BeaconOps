# PCB

这里放的是 BeaconOps 的 PCB 工程源文件，就一份：`ProPrj_BeaconOps.epro`，嘉立创专业版做的。

## 关于这块板

这块板是去年画的。主芯片 ESP32-C3 是 QFN-32，旁边的外置 flash Winbond W25Q128JV 是 WSON-8，这两颗还算正常。难的是两颗无引脚器件：

- 姿态感知 ST LSM6DS3TR-C 是 LGA-14，14 个焊盘全在芯片底部，上板后看不见焊点，只能靠贴片机 / 热风台的位置精度，偏 0.1 mm 就可能虚焊
- 音频功放 MAX98357AEWL+ 是 WLP-9（1.34×1.34 mm），芯片裸片 + 9 颗焊球的 BGA 类封装，返修完全依赖 X-Ray，手工场景下出问题基本只能换一颗

被动器件清一色 0402，最窄器件间距约 0.4 mm。整体元器件挨得比较近。**这块板的复刻门槛主要不在文件本身，而在你手上的活**——能不能稳定回流、能不能贴正，要看你自己的设备和经验。

我没有特意把它改成更“友好”的版本，因为 BeaconOps 本来就不是“随便焊一下就能玩”的板子。

## 怎么打开

- 用嘉立创专业版打开 `.epro` 工程
- 打开后能看到原理图、PCB、封装和工程设置
- 想要 Gerber、BOM、坐标文件，自己在工程里重新导出就行，仓库里不重复带一份
